# 01 - Введение

`scrapedoctl` — это профессиональная CLI-утилита и сервер Model Context Protocol (MCP), разработанный для обеспечения удобного взаимодействия AI-агентов с веб-контентом. Используя API [Scrape.do](https://scrape.do/), проект предоставляет мощный движок для скрапинга с обходом анти-бот систем и рендерингом JavaScript, оптимизированный как для человека, так и для машин.

## Цель проекта

Основная цель `scrapedoctl` — предоставить AI-агентам (таким как Claude Code, Gemini CLI и др.) высококачественные Markdown-представления веб-страниц, минимизируя при этом затраты за счет персистентного кеширования и эффективного управления запросами.

## Основные возможности

- **Интерактивный REPL**: Полноценная оболочка для ручного исследования страниц.
- **Персистентное кеширование**: Встроенное хранилище SQLite для экономии токенов и ведения истории.
- **Интерфейс для машин**: Полная поддержка MCP и JSON-метаданных для динамического обнаружения инструментов.
- **Обход анти-ботов**: Использование ротации прокси и браузерного рендеринга Scrape.do.
- **Современная архитектура**: Написано на Go 1.26, ядро без внешних зависимостей, строгое соблюдение стандартов линтинга.
