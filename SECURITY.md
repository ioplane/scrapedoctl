# Security Policy

## Supported Versions

Only the latest version of `scrapedoctl` is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it by opening an issue or contacting the maintainers directly.

We take all security reports seriously and will work to resolve the issue as quickly as possible.
